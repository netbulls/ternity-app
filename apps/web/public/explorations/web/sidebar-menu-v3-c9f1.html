V3 — The Popover

Org-first identity block — the org name is the primary line, person name secondary. Rounded-square avatar instead of circle. Click opens a floating popover with an arrow pointing to the user block.

User sees User

  • Settings (with , shortcut hint)
  • Sign out

Admin sees Admin

  • Admin group: People, Projects
  • Settings
  • Sign out

Design notes

  • Org name first — future org switcher slot (like Claude's workspace)
  • Rounded-square avatar — distinguishes from circular user avatars elsewhere
  • No chevron — the whole block is the click target, clean look
  • Popover floats above with a subtle arrow, doesn't cover the user block
  • Keyboard shortcut hints shown inline (muted, right-aligned)
  • Grouped menu items with labeled admin section
  • Wider popover (200px) than V1 dropdown — gives items room to breathe
