Design Notes — V2 The Transfer List

  • Classic enterprise pattern — instantly recognizable to users familiar with admin tools.
  • Explicitly shows both available AND assigned — no ambiguity about what's unassigned.
  • Bulk actions (assign all / remove all) make large batch operations fast.
  • Each panel is independently searchable — scales well for orgs with many projects.
  • Takes up significant horizontal space — two full panels side by side.
  • Multi-step: select items, then click arrow. More clicks than a simple checkbox.
  • Role badge toggle is subtle and might be missed — it's inline on each assigned item.
  • Pattern can feel "enterprise heavy" for a modern product — less friendly than alternatives.
