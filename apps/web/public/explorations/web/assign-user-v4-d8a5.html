V4 — The Inline Table

A data table approach integrated into the user detail page. Every project in the organization is listed as a row, with toggle switches for instant assignment control.

Pros

  • Most data-dense — see all projects at a glance in a familiar table layout
  • Sortable columns make it easy to find projects by name, client, or role
  • Filter + "assigned only" checkbox quickly narrows the view
  • Toggle switches provide instant, single-click assignment changes
  • Bulk actions allow assigning/removing many projects at once via checkboxes
  • Role dropdown is contextually enabled — only when the project is assigned
  • Tab navigation integrates naturally into a user profile page

Cons

  • Can feel overwhelming with dozens of projects — all visible simultaneously
  • No visual grouping by client — relies on sorting to cluster related projects
  • Toggle + checkbox in same row may cause confusion (one for assignment, one for bulk select)
  • Less visual richness — no project colors or metadata beyond name/client/role
  • Horizontal space usage is suboptimal (wide table, mostly empty cells)

Best for

  • Organizations with many projects (20+) where scanning efficiency matters
  • Admins who prefer data-table workflows and keyboard-driven interaction
  • Situations where bulk assignment operations are common (onboarding new team members)
