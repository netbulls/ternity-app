V1 — The Dropdown

Classic dropdown pattern. The user block at the bottom of the sidebar is clickable. A menu pops up above it with role-appropriate items.

User sees User

  • Settings (theme, profile, integrations)
  • Sign out

Admin sees Admin

  • Admin section: People, Projects
  • Settings
  • Sign out

Design notes

  • Chevron (chevrons-up-down) signals clickability
  • Menu opens upward (bottom-anchored sidebar)
  • Admin items grouped under a label
  • Sign out is visually destructive (red)
  • Org name shown below user name — future org switcher slot
  • No Settings in sidebar nav — it lives in the menu
