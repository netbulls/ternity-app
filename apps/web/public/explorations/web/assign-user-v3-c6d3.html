Design Notes — V3 The Tag Cloud

  • Most compact layout — all projects visible at a glance without scrolling.
  • Visual distinction is immediate: filled teal = assigned, dashed outline = available.
  • Single click to toggle — fastest interaction model of all three variants.
  • Floating count badge gives instant summary without scanning individual items.
  • Client grouping with labels provides context without heavy UI chrome.
  • Role selector is tiny and easy to miss on each pill — limited space for metadata.
  • Pill layout reflows unpredictably at different widths — less structured than a list.
  • Doesn't scale well past ~20 projects — pill cloud becomes visually noisy.
  • No search/filter — relies on the compact view being sufficient. Would need search for larger lists.
