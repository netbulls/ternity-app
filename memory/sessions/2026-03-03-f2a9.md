## Task: Create design lab explorations for user-to-project assignment

Created 2 static HTML exploration files for the Ternity design lab exploring UX approaches for assigning projects to a user from a User Management / User Profile admin page.

### Files Created

1. `assign-user-v4-d8a5.html` — **The Inline Table**: Data table with toggle switches per row, sortable columns, search/filter, bulk actions bar. User profile header at top with tab navigation. Most data-dense approach.
2. `assign-user-v5-e9b2.html` — **The Accordion**: Client-grouped collapsible sections with 2-column project card grids. Color bars, checkboxes, role pill selectors. "Select All" per client. Summary bar at bottom with running totals.

### Details

- Both use the standard exploration boilerplate (themes.css, lucide icons, theme/scale switchers)
- Fictitious user: Elena Marsh (v4), James Oakley (v5)
- 8 projects across 4 clients (Acme Corp, Stellar Labs, NovaTech, BluePeak)
- All interactive: toggles, checkboxes, role selectors, search filtering, accordions, sorting, bulk actions
- Design notes sections with pros/cons/best-for analysis at bottom of each file
