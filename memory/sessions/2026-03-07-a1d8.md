## Key Decisions

1. **Column widths preserved via `<colgroup>`**: Since `<th style="width:...">` was defining column widths in the thead, removing thead would cause columns to auto-size. Added `<colgroup><col style="width:...">` to maintain the same layout.

2. **V6 summary/appendix headers kept**: The user's complaint was about the data table column headers that repeat on every page. The Summary and Appendix tables in V6 are small reference tables — their headers are essential for readability and aren't the same issue.

3. **Dead code cleaned up**: V7's `renderUserSection()` is never called (leftover from before continuous-flow refactor) but its thead was cleaned up for consistency.
