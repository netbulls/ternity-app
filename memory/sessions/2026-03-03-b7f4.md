# Session 2026-03-03-b7f4

## Task: Create project member assignment design explorations

Created 3 static HTML exploration files for the Ternity design lab exploring UX for assigning users/members to a project from a Project Settings admin page.

### Files Created

1. `apps/web/public/explorations/web/assign-project-v1-f3c8.html` — **The Member List**: Straightforward list with inline search-to-add panel. Shows member count, avatars, role badges, remove buttons.
2. `apps/web/public/explorations/web/assign-project-v2-g5d9.html` — **The Avatar Grid**: Visual avatar grid with teal ring borders for assigned, dimmed for unassigned. Click-to-toggle assignment. Role indicator dots. Grid/list view toggle.
3. `apps/web/public/explorations/web/assign-project-v3-h7e2.html` — **The Split Panel**: Split layout with project info panel (left) and member management (right). Combobox search for adding, inline role dropdowns, drag-to-reorder hints.
