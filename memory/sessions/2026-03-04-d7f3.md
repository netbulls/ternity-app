# Session 2026-03-04-d7f3

## Goal

Add a new theme inspired by Claude's UI — warm beige/cream tones, earthy palette — but without serif fonts (keeping the project's Oxanium sans-serif). The user wants something simpler.

## Context

- Theme system has 6 themes already (3 dark, 2 light, 1 high-contrast dark)
- Adding a theme requires: 1) ID + meta in themes.ts, 2) CSS block in themes.css
- Claude's UI palette: warm cream/beige background (~#FAF7F2), dark brown text, terracotta/rust accent, soft muted tones
- All themes must use the same sans-serif Oxanium font — no serif allowed

## Decisions

- Named it "Sandstone" (user chose from Parchment/Linen/Sandstone)
- Kept Electric Teal as primary for functional consistency (timers, labels, avatars)
- Used terracotta (#C27949, hsl 22 50% 53%) as chart-2 and label-2 color for earthy contrast
- Warm cream background (~37 33% 93%), slightly lighter cards (~38 33% 96%)
- Deep warm brown text (~30 5% 17%), medium brown muted text (~20 5% 50%)
- Warm destructive red (~10 50% 53%) instead of the bright red used in other themes
